# Student Management System in C

A menu-driven Student Management System developed in C using
structures and file handling for permanent data storage.

This project demonstrates real-world use of C programming concepts
such as structures, arrays, file handling, and modular programming.

---

## 🚀 Features

- Add student records
- View all students
- Search student by roll number
- Update student details
- Delete student records
- Permanent storage using binary files

---

## 🛠 Technologies Used

- C Programming
- Structures
- File Handling (`fopen`, `fread`, `fwrite`)
- Modular Functions
